# 🔨 モグラたたき (Whack-a-Mole)

飛び出してくるモグラをすばやくたたいて高スコアを狙うゲーム。

## 遊び方
1. 難易度を選んでスタート
2. 穴から出てきたキャラクターをクリック/タップ
3. 30秒で何点取れるか競う！

## キャラクター
| キャラ | ポイント | 説明 |
|---|---|---|
| 🐭 | +10 | 通常のモグラ |
| 🐹 | +20 | 素早いモグラ |
| 💎 | +50 | レアなダイヤ |
| 💣 | -30 | 爆弾！たたかないで |

## 難易度
| 難易度 | 出現時間 | スポーン間隔 |
|---|---|---|
| かんたん | 1.8秒 | 2秒 |
| ふつう | 1.2秒 | 1.4秒 |
| むずかしい | 0.8秒 | 1秒 |

## 特徴
- コンボシステム（連続ヒットでスコア倍増、最大×5）
- 4種類のキャラクター（爆弾注意！）
- ミスカウント
- ヒットエフェクト（スコアが浮かび上がる）
- 3段階の難易度

## 技術スタック
HTML / CSS / Vanilla JavaScript
